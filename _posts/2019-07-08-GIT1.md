## Git 초기 설정

​

1. Git bash 설치

2. 커맨드 창 Git 설치 유무 확인
```
$ git --version
```

3. global하게 사용될 github 메일, 이름 등록
```
$ git config --global user.name "username"
$ git config --global user.email "useremail"
```
​



​

​

## 유용한 명령어

```
$ cd 파일경로 //위치 이동 명령어
$ mkdir 폴더 이름 //디렉터리, 폴더 생성 명령어
$ git status // 변경 사항 확인 명령어
$ git diff // 수정된 내용 확인
$ echo "banana" > readme.txt // banana라는 내용을 readme.txt에 담에 생성
$ atom README.md // atom 편집기로 README.md 파일 열기
```
