# Git 원격 브랜치 삭제 방법

- 삭제할 브랜치 위치에서 진행

```
git push origin --delete 브랜치 이름
```
<br/>

# Git 로컬 브랜치 삭제 방법

- 소스트리 활용법
- 삭제할 브랜치가 아닌 위치에서 삭제할 브랜치를 오른쪽 마우스 클릭
- 삭제 클릭
- 강제 삭제 체크박스 체크
- 삭제 완료
