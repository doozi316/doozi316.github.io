## 웹서버란?

​

- 웹 서버는 소프트웨어(Software)를 보통 말하지만, 웹 서버 소프트웨어가 동작하는 컴퓨터를 말함

- 웹 서버의 가장 중요한 기능은 클라이언트(Client)가 요청하는 HTML 문서나 각종 리소스(Resource), 데이터를 전달하는 것

- 웹 브라우저나 웹 크롤러가 요청하는 리소스는 컴퓨터에 저장된 정적(static)인 데이터이거나 동적인 결과가 될 수 있음

​

_용어정리)_

- 정적 데이터 : 보통 이미지, HTML 파일, CSS 파일, Javascript 파일과 같이 컴퓨터에 저장되어 있는 파일

- 동적 결과 : 웹 서버에 의해서 실행되는 프로그램을 통해 만들어진 결과물

- 웹 크롤러 : 네이버나 구글 같은 검색 사이트에서 다른 웹 사이트 정보를 읽어갈 때 사용하는 소프트웨어

​

웹 서버는 동시에 요청한 여러개의 요청을 받아들여서 그 결과를 브라우저에 전송.

해석한 HTML 문서와 읽어들인 여러개의 응답을 하나로 합쳐서(렌더링) 화면에 결과가 보이게 함.

​

​

​

​

### 웹 서버 소프트웨어의 종류

​

​

-가장 많이 사용하는 웹 서버는 Apache, Nginx, Microsoft IIS

-Apache웹 서버는 Apache Software Foundation에서 개발한 웹서버로 오픈소스 소프트웨어(Open-source Software)이며, 거의 대부분 운영체제에서 설치 및 사용을 할 수 있음

-Nginx는 차세대 웹서버로 불리며 더 적은 자원으로 더 빠르게 데이터를 서비스하는 것을 목적으로 만들어진 서버이며 Apache웹 서버와 마찬가지로 오픈소스 소프트웨어
