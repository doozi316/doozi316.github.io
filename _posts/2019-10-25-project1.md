위 두 사이트를 참조해서 VSCode에 spring boot 설치 해주었다.

Java Extension Pack과 Spring Boot Extension Pack 확장 프로그램을 설치하고 Maven 프로젝트를 생성했다.

튜토리얼 과정 중 mvn clean install 을 진행하던 중, 아래와 같은 오류를 마주했다.

**<u>"Failed to execute goal org.apache.maven.plugins:maven-clean-plugin:2.5:clean (default-clean)"</u>**

이는 ```mvn clean install -Dmaven.clean.failOnError=false``` 이라고 입력했더니 해결되었다. ^^

​

원인은 user settings에 java.home의 경로에 오타가 있었기 때문이었다. ^^....
