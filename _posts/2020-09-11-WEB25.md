# Servlet이란?

- 클라이언트와 서버의 요청과 응답을 처리하는 JAVA의 **클래스**
- Servlet은 모든 유형의 요청에 응답할 수 있지만 일반적으로 웹 애플리케이션에서 사용된다.
- 이러한 웹 어플리케이션의 경우, Java Servlet 기술은 HTTP 특정 서블릿 클래스를 의미한다.
- `javax.servlet`과 `javax.servlet.http` 패키지는 Servlet 작성에 필요한 인터페이스와 클래스를 제공한다.
- 모든 서블릿은 생명주기 메서드라고 불리는 서블릿 인터페이스를 구현해야한다.
- 일반적인 서비스를 구현할 때, Java Servlet API를 제공하는 `GenericServlet` 클래스를 사용하거나 `extend` 할 수 있다. 
- `GenericSevlet` 은 HTTP 서비스를 다루는 `doGet` 이나 `doPost` 같은 메소드를 제공한다.
