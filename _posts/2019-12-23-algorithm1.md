# 10171번


![](https://doozi316.github.io/assets/images/Algorithm1/3.png)
```
public class Main{
    public static void main(String args[]){
        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");                   
    }
}
```
* 출력이 되지 않는 특수기호를 출력하려면 역슬레쉬(\)를 특수기호 앞에 입력해 줘야함.
