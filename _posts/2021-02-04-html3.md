# Template 태그

- 페이지를 불러온 순간 즉시 그려지지는 않지만, 이후 JavaScript를 사용해 인스턴스를 생성할 수 있는 HTML 코드를 담는 방법
- 콘텐츠 조각을 나중에 사용하기 위해 담아놓는 컨테이너
- 페이지를 불러오는 동안 구문 분석기가 템플릿 태그의 콘텐츠도 읽기는 하지만, 이는 유효성 검증을 위함이며 렌더링 하기 위함이 아니다.
